Functions:
User is able to
 - Login (Will not be authenticated if password fails)
 - View Gallery (With each order made, the cart will be updated)
 - View purchases (even in the purchase page, user is able to modify the number of items he wants
 - View Cart (This happens after checkout, and object is added to the database)
 - Logout (Once session Id is expired, user is not able to use the same session Id to open the URL again.)

In this repository,
two versions of the shopping cart is uploaded.
shopping cart could be done either using
session object or a key-value dictonary
